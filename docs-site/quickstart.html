🚀 Quickstart

Get up and running with A2Apex in under 5 minutes. This guide will walk you through installing the SDK, validating your first agent card, and running your first test suite.

💡 Prerequisites

You'll need Python 3.10+ installed. You can also use the web interface if you prefer not to install anything.

1

Install the SDK

Install A2Apex from PyPI:

pip install a2apex

For rich terminal output with colored tables:

pip install a2apex[rich]
2

Validate Your First Agent Card

Create a Python script to validate an agent card:

from a2apex import A2ApexClient

# Create a client
client = A2ApexClient()

# Validate from URL (fetches /.well-known/agent-card.json)
report = client.validate_card("https://your-agent.example.com")

# Check the results
print(f"✅ Valid: {report.is_valid}")
print(f"📊 Score: {report.score}/100")
print(f"❌ Errors: {report.error_count}")
print(f"⚠️ Warnings: {report.warning_count}")

# Show any errors
if report.errors:
    print("\nErrors to fix:")
    for error in report.errors:
        print(f"  ✗ {error.field}: {error.message}")
        if error.suggestion:
            print(f"    💡 {error.suggestion}")

Or validate a dict directly (no HTTP):

from a2apex import validate_agent_card

card = {
    "name": "My Agent",
    "url": "https://api.example.com/a2a",
    "version": "1.0.0",
    "capabilities": {"streaming": True},
    "skills": [
        {
            "id": "chat",
            "name": "Chat",
            "description": "General conversation"
        }
    ]
}

report = validate_agent_card(card)
print(report)  # ✓ Valid (Score: 75/100) — 0 errors, 5 warnings
3

Run Your First Test

Run the full compliance test suite against your agent:

from a2apex import A2ApexClient

client = A2ApexClient()

# Run all tests
results = client.test_agent("https://your-agent.example.com")

print(f"\n📊 Compliance Score: {results.score}/100")
print(f"✅ Passed: {results.passed}/{results.total_tests}")
print(f"❌ Failed: {results.failed}")
print(f"⚠️ Warnings: {results.warnings}")

# Show individual test results
print("\nTest Results:")
for test in results:
    icon = "✅" if test.passed else "❌"
    print(f"  {icon} {test.name}: {test.message}")

Validate State Transitions

A2A defines a strict state machine for task lifecycle. Validate your transitions:

from a2apex import validate_transitions, is_terminal_state

# Check if a sequence of state transitions is valid
result = validate_transitions(["submitted", "working", "completed"])
print(f"Valid: {result.is_valid}")  # True

# Invalid transition (can't go from completed to working)
result = validate_transitions(["submitted", "working", "completed", "working"])
print(f"Valid: {result.is_valid}")  # False

for violation in result.violations:
    print(f"  ✗ {violation.from_state} → {violation.to_state}")
    print(f"    {violation.reason}")

# Check terminal states
print(is_terminal_state("completed"))  # True
print(is_terminal_state("working"))    # False

Using Async

All methods have async versions for use with asyncio:

import asyncio
from a2apex import A2ApexClient

async def main():
    client = A2ApexClient()
    
    # Async validation
    report = await client.avalidate_card("https://agent.example.com")
    print(f"Score: {report.score}/100")
    
    # Async testing
    results = await client.atest_agent("https://agent.example.com")
    print(f"Passed: {results.passed}/{results.total_tests}")

asyncio.run(main())

CI/CD Integration

Add A2Apex to your CI/CD pipeline to catch compliance issues early:

# test_a2a_compliance.py
from a2apex import A2ApexClient
import pytest

def test_a2a_compliance():
    client = A2ApexClient()
    results = client.test_agent("https://your-agent.example.com")
    
    # Fail if compliance score is below 80
    assert results.score >= 80, f"A2A compliance too low: {results.score}/100"

def test_critical_endpoints():
    client = A2ApexClient()
    results = client.test_agent("https://your-agent.example.com")
    
    # Check specific critical tests
    critical_tests = ["agent_card_fetch", "message_send"]
    for test in results:
        if test.name in critical_tests:
            assert test.passed, f"Critical test failed: {test.name}"

GitHub Actions Example:

name: A2A Compliance

jobs:

    runs-on: ubuntu-latest

        with:
          python-version: '3.11'
      - run: pip install a2apex pytest
      - run: pytest test_a2a_compliance.py

Using the Web UI

Don't want to install anything? Use the A2Apex web interface:

  1. Go to a2apex.io
  2. Enter your agent's URL
  3. Click "Test Agent"
  4. View your compliance score and detailed results

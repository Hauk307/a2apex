What is A2Apex?

A2Apex is the testing toolkit for the A2A (Agent-to-Agent) protocol — Google's open standard for AI agent interoperability. Think of it as "Postman for AI Agents."

With A2Apex, you can:

  • Validate Agent Cards — Ensure your agent's metadata conforms to the A2A specification
  • Test Endpoints — Run live tests against your A2A agent's JSON-RPC endpoints
  • Check Compliance — Get a 0-100 compliance score with detailed reports
  • Automate Testing — Integrate into CI/CD pipelines with the Python SDK

Quick Example

Validate an agent card and run tests with just a few lines of Python:

from a2apex import A2ApexClient

client = A2ApexClient()

# Validate an agent card
report = client.validate_card("https://your-agent.example.com")
print(f"Score: {report.score}/100")
print(f"Valid: {report.is_valid}")

# Run full test suite
results = client.test_agent("https://your-agent.example.com")
print(f"Passed: {results.passed}/{results.total_tests}")

Key Features

Feature Description
Agent Card Validation Full schema validation against A2A spec v0.3
Live Endpoint Testing Test message/send, tasks/get, tasks/cancel
State Machine Validation Verify task lifecycle state transitions
Streaming Tests SSE event ordering and format validation
Auth Testing API key, OAuth2, JWT scheme validation
Error Handling JSON-RPC 2.0 error compliance testing
Performance Testing Latency, concurrency, cold start tests
Compliance Scoring 0-100 score with detailed breakdown
